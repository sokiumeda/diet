# diet

ダイエット管理

自分用の体重・食事記録アプリ（PWA）です。単一ファイル `diet-tracker-pwa.html` にHTML/CSS/JSがすべて内包されており、外部サーバーは使わずブラウザの localStorage だけにデータを保存します。

公開URL: https://sokiumeda.github.io/diet/

## 使い方

- 「記録」タブで日付・体重・ジムに行ったかを入力して記録を追加
- 「推移」タブで体重の推移グラフを確認
- 「食事例」タブでその日のおすすめ献立とカロリー/タンパク質の目安を確認
- ホーム画面に追加（iOSのSafari共有メニューなど）するとアプリのように起動でき、オフラインでも開けます

## データはローカル保存です

記録データはブラウザの localStorage のみに保存されます。サーバーには送信されません。ブラウザのデータ削除や、iOSでの長期未使用によるアプリデータの自動消去などで記録が失われる可能性があるため、「記録」タブ下部の「データ管理」から定期的に **JSONエクスポート** して手元に保管することをおすすめします。機種変更やデータ消失時は、同じ画面から「JSONからインポート」で復元できます（インポートは既存データを上書きします）。

## 開発メモ

- `diet-tracker-pwa.html`: アプリ本体（単一ファイル）
- `manifest.json` / `sw.js` / `icons/`: PWA化（ホーム画面追加・オフライン起動）のための最小構成
- `.github/workflows/deploy-pages.yml`: `main` への push で GitHub Pages に自動デプロイ（`diet-tracker-pwa.html` を `index.html` として配信）
